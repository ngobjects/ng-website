ng-objects! alpha

A free, open source, Java-based web development framework inspired by the design and principles of Apple's WebObjects. It's compatible with WebObjects deployment environments, allowing you to seamlessly deploy ng-objects applications to existing WO environments.

Quick start GitHub

Our status

ng-objects is under development. What we currently have working is basic templating and routing, with component actions being under development.

We plan to be ready for production use around the release of Java's next planned LTS release (21) in September 2023.

On the whole things look promising, but be sure that this is a long-term project. We want to get things as right as possible before we start "blowing our horns" so to speak, to minimize "API flux" once the libraries see real world use.

Oh, and of course this website runs on ng-objects and it's source is on github.
